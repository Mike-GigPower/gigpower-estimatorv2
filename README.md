# GigPower Quote Estimator (web app)

Next.js + TypeScript quote estimator recreated from the Excel template.

## Run
```bash
npm install
npm run dev
```

## Print
Use the **Print / PDF** button to print a customer-facing quote; blank non-labour rows are hidden on print.

## Codex project management
This repo includes an `AGENTS.md` playbook plus `PROJECT_TASKS.md` so Codex can track intake, active work, blockers, and completed tasks in one place.
